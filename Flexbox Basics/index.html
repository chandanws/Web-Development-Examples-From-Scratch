Flex-Wrap

By default, flexbox will try to fit all of the flex items on a given line. Even though the items below have a width set to 500px in the css, they are staying on the same line. We can define how wrapping works with flexbox by choosing one of the following flex-wrap properties: nowrap(default), wrap, wrap-reverse.
